Descrição do Projeto

O Equestria RPG & Ponytopia é um projeto web inspirado no universo de My Little Pony. O sistema permite que o usuário realize cadastro e login, participe de um teste de personalidade, receba uma carta mágica personalizada, explore a Ponytopia, resgate uma recompensa diária e acompanhe seu álbum de figurinhas, onde o progresso é salvo pelo sistema.

Funcionalidades

* Cadastro de usuários.
* Login de usuários.
* Teste de personalidade.
* Carta mágica personalizada.
* Ponytopia.
* Recompensa diária (uma vez por dia).
* Álbum de figurinhas com progresso salvo.

Tecnologias Utilizadas

* HTML
* CSS
* JavaScript

Estrutura do Projeto (MVC)

* Model: responsável por armazenar e gerenciar os dados, como cadastro de usuários, recompensa diária e progresso do álbum de figurinhas.
* View: responsável pela interface apresentada ao usuário.
* Controller: responsável por receber as ações do usuário, processar as informações e conectar a View ao Model.
